LostInTheCave

Lost In The Cave é um jogo de aventura 2D desenvolvido em C++ utilizando os princípios de Programação Orientada a Objetos (POO). O jogo se passa em um ambiente misterioso de uma caverna, onde o objetivo é navegar por obstáculos, evitar inimigos e encontrar a saída. O jogo suporta modos de um jogador e dois jogadores, e possui dois níveis distintos com personagens, obstáculos e níveis de dificuldade variados. O projeto foi desenvolvido como parte do curso de Técnicas de Programação.

Scanner 3D

'Scanner 3D para Cálculo de Volume de Objetos' é um projeto voltado para determinar a metodologia mais eficaz de cálculo de volume de objetos. O sistema utiliza um sensor infravermelho para capturar as distâncias entre o sensor e o objeto, gerando uma nuvem de pontos que representa a superfície do objeto. As metodologias propostas para o cálculo de volume são baseadas em algoritmos e cálculos geométricos derivados dos dados coletados.

Teste Matemático

O objetivo do projeto é disponibilizar um simulado de questões de matemática para estudantes do ensino fundamental, que não somente estimula seus aprendizados matemáticos, mas também melhora o raciocínio lógico e a interpretação de problemas.
Todas as questões foram inspiradas em problemas matemáticos que a minha irmã de 11 anos, que está fazendo cursinho para entrar no Colégio Militar de Curitiba, pedia ajuda.
Dentro do site você pode relizar a prova, verificar a sua nota e ver a resposta e explicação de cada questão ao finalizar. Esse foi o meu primeiro projeto utilizando HTML, CSS e JavaScript.
